How to Declare a variable?

  1. var (global scope -> attach itself window)
  2. let (local scope -> { }, script block )
  3. const (local + readonly + initilization only)

How to spot syntax and its meaning?

  1. [] -> Array
  2. { } -> object, control flow, block code
  3. () -> methods, function3
  4. leftItem : rightItem -> css class, json

Array functions

  1. pop() - remove last element (nth element, n = length) <-> shift() remove first element (0 index element)
  2. push() - add element (nth element, n = length) <-> unshift() add element (0th index,at the very begining)
